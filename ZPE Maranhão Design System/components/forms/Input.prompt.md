Campo de texto com label monoespaçado e anel de foco azul institucional. Use em formulários de contato e cadastro de investidor.

```jsx
<Input label="E-mail corporativo" type="email" placeholder="nome@empresa.com" />
<Input label="CNPJ" error="CNPJ inválido" />
```

Props: `label`, `hint`, `error` (borda vermelha + mensagem). Aceita todos os atributos de `<input>`.
