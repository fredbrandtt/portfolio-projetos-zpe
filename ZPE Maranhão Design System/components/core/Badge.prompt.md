Pílula de metadado ou status em JetBrains Mono. Use para rótulos institucionais, status de processos e tags curtas.

```jsx
<Badge variant="green">Ativo</Badge>
<Badge variant="blue">Institucional</Badge>
<Badge variant="warning">Em análise</Badge>
```

Variantes de marca: `green`, `blue`. Semânticas: `success`, `warning`, `error`, `info`. Neutra: `outline`. Texto curto; evite frases.
